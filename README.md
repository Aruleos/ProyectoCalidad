# Calidad

Proyecto de calida y pruebas de software que utiliza la API de pokémon y muestra distintas opciones:
  1. Buscar Pokémon
  2. Guardar Pokémon
  3. Mostrar Pokémon
  4. Eliminar Pokémon
  5. Buscar Equipo
  6. Mostrar Equipo
  7. Guardar Equipo
  8. Eliminar Equipo
  0. Salir
